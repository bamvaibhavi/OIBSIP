# Task 4: Login Authentication
A secure login system that allows users to register and access a private dashboard.

## Features
* **User Accounts:** Functionality for users to register and login.
* **Secured Access:** Successful login redirects users to a "Secured Dashboard".
* **Session Control:** Includes a logout feature to secure the user's session.
* **Validation:** Built using a programming language of choice (HTML/CSS/JS) to manage user authentication.
